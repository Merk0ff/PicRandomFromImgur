# PicRandomFromImgur

Фановый мини проект.

## Установка

### Unix
```
python3 setup.py install
```
### Windows
```
python setup.py install
```
или
```
python.exe setup.py install
```
## Запуск

### Unix
```
python3 ./src/main.py
```
### Windows
```
python ./src/main.py
```
или
```
python.exe ./src/main.py
```
